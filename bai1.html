    • I. HTML là gì?

    HTML là chữ viết tắt của Hyper Text Markup Language, dịch ra có nghĩa là Ngôn ngữ đánh dấu siêu văn bản. Trong lập trình web, HTML có nhiệm vụ là xây dựng lên bố cục trang web.

    Mỗi một website được cấu tạo nên từ nhiều trang web, ví dụ như phambinh.net là một website và mỗi trang con trên phambinh.net là một trang web. Mỗi trang web đều xây dựng bố cục dựa trên HTML, và mỗi trang web như vậy được gọi là một tài liệu HTML (nhiều lúc mình gọi là file html hoặc tệp tin html).

    HTML là ngôn ngữ tối cơ bản để xây dựng lên một trang web, vì vậy mà biết HTML là điều kiện cần nếu bất kỳ ai muốn trở thành một web developer. Ngay cả một số bạn làm bên marketing online hoặc biên tập nội dung website đôi khi cũng cần phải biết tới HTML.

    II. HTML có thể làm được gì?

    HTML có thể sử dụng để xây dựng bố cục trang web, cụ thể bố cục trên trang web là:

  • ♥ Các thành phần trên trang web: Văn bản, hình ảnh, các đường links, video, audio,…
  • ♥ Cách sắp xếp các thành phần kể trên.
    • Vậy với HTML bạn có thể sử dụng để chèn các đoạn văn bản, hình ảnh, link, video,.. lên trang web và sắp xếp chúng hợp lý theo ý bạn.

    III. Một tài liệu HTML hoạt động thế nào

    Khi bạn truy cập vào một trang web, trình duyệt sẽ nhận về mã HTML của trang web đó. Sau khi nhận được mã HTML, trình duyệt sẽ biên dịch những mã HTML đó sang giao diện website để người dùng có thể đọc hiểu.

    Để hình dung dễ hơn, bạn có thể xem hình ảnh phía dưới.

    Trong ảnh trên, phía bên trái chính là mã HTML chưa được biên dịch – cái mà trình duyệt sẽ nhận được khi truy cập vào mỗi trang web. Còn phía bên phải là kết quả sau khi trình duyệt biên dịch đoạn mã HTML nhận được thành giao diện website.

    Ở dạng nguyên thể, HTML chỉ là văn bản đơn thuần, chỉ khi đi qua bộ biên dịch của trình duyệt nó mới có thể trở thành như những gì bạn trông thấy.

    IV. Sử dụng phần mềm gì để viết HTML

    Có rất nhiều phần mềm trên máy tính giúp bạn có thể viết được mã HTML, nhưng mình gợi ý tới bạn phần mềm Sublime Text 3, visual code.

    V. Sử dụng phần mềm gì để chạy HTML

    Về cơ bản bạn sẽ sử dụng một trình duyệt để chạy HTML, mà trình duyệt thì đương nhiên máy tính nào cũng có. Bạn chỉ cần mở tệp tin HTML bằng trình duyệt là có thể thấy được kết quả.

    Hiện nay có rất nhiều trình duyệt nổi tiếng, tuy nhiên mình gợi ý các bạn nên sử dụng Google Chrome, cốc cốc, edge, .

    VI. Các phiên bản HTML

    Từ khi khai sinh cho tới giờ HTML đã trải qua khá nhiều phiên bản, cụ thể bạn có thể theo dõi trong bảng sau:

    Version Năm ra đời
    HTML 1991
    HTML 2.0 1995
    HTML 3.2 1997
    HTML 4.01 1999
    XHTML 2000
    HTML5 2014

    Phiên bản HTML mới nhất cho tới thời điểm mình viết bài viết này (2019) là phiên bản HTML5. Và trong seri này mình cũng sẽ hướng dẫn các bạn theo cú pháp của HTML 5.
